California Job Case is a set of typographic tools for making your fonts work rhythmically with using a base line grid. It is currently in alpha and is working well in Chrome, but not some much in Safari (my favourite). It's based on SASS so please be sure you have that installed. Future releases will be installable via bower and a gem. The GitHub repo is right here.

The basic premiss of this framework is to set a few variables and then get font-size, line-height, padding and margins from some behind the scenes calculations. The base variables are the base font size and the line height you wish to use. From that use classes like font-plus-1 (with options for 1-4) or font-minus-1 (with options to 1-2). These are rhythmically extrapolated which you can extend further using the get-font-and-line($multiple[int], $larger[boolean]) mixin. There is also a the rhythm-spacer($multiple) which is will return a value to use paddings an margins in multiples of half line heights.

One of the reasons I saw a need for this is that html that by default bigger means more important, and as a designer that is just not true. Placement, font style and colour can be just as important in the factors of informing the user's eye of the hierarchy.
